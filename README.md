# hc-cache-plugin

Redis and Memcached integration for the Apito API platform.

## Features

- Redis integration
- Memcached support
- In-memory caching
- Cache invalidation
- Performance metrics

## Installation

```bash
go build -o hc-cache-plugin .
```

## Configuration

See `config.yml` for env vars: `CACHE_PROVIDER`, `REDIS_URL`.
